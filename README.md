# 🔐 Simple Crypto Wallet (Testnet)

A basic Ethereum wallet that can generate addresses, check balance, and send ETH using testnet.

## 🚀 Features
- Generate wallet
- View address & private key
- Check balance
- Send ETH (Sepolia testnet)

## ⚠️ Warning
This is for learning purposes only.
Do NOT use real funds.

## 🖥️ How to Run
Open index.html in browser

## ⚙️ Tech
- HTML, CSS, JS
- Ethers.js
